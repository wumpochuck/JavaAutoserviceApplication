## Requirements
- javafx 17.0.11
- mySql connector 8.3.0
- JDK 21

## Usage
  Run "DatabaseLauncher", then run "Main" with VM options:
  ``` powershell
  --module-path
  path/to/javafx/lib/
  --add-module
  javafx.controls,javafx.fxml
  ```
